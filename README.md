# Java_Project
Color Match Game
A fun, timed Java Swing game that tests your reaction speed and color-word association skills. Match the displayed color name by clicking the button with the matching ink color (not the word on the button). Beat your high score in 60 seconds!

Objective: Click the button whose color matches the prompt word's ink color (e.g., if "Red" appears in blue ink, click the red button).
